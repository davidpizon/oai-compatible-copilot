Agentic Router Configuration

This extension exposes a single model to Copilot Chat and forwards every request to the endpoint below.

Connection

The Agentic Router (or any OpenAI-compatible) endpoint requests are sent to.

Model

Name shown in the Copilot model picker. When empty, the picker shows the model id from the totallyhot.spark.modelId setting (default: agentic-router), edited in settings.json.
Protocol used to talk to the endpoint.

Advanced

Minimum delay between consecutive requests.
Lines to read with the read_file tool (0 = let the model decide).

Git Commit

Language used for generated commit messages.
